### Cron Jobs:
Son conocidas popularmente como tareas programadas, en teoria son funciones que se ejecutan basadas en una expresion de tiempo (cron expression).

### Cron Expressions:
Son cadenas de texto que expresan tiempo, por ejemplo: 
```* * * * *```
